# React Globules

[![npm package][npm-badge]][npm]

React JS component that gives your project a cool interactive background

## Properties

```html
<Globules
    widthPercent={true} 
    heightPercent={true} 
    width={100} // % or px
    height={100} // % or px
    speed={4} // 0 - 9
    density={50} // 0 - 99
    color={'#ffffff'} // 6 char long hex
    background={'#000000'} // 6 char long hex
>
    <p>Test</p>
</Globules>
```
